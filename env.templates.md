# Environment Configuration Templates

Copy the appropriate template file to `.env` for each environment.

## Development
Copy `.env.dev.example` to `.env` for development environment.

## Staging
Copy `.env.stg.example` to `.env` for staging environment.

## Production
Copy `.env.prod.example` to `.env` for production environment.

## Quick Setup
```bash
# For development
cp .env.dev.example .env

# For staging
cp .env.stg.example .env

# For production
cp .env.prod.example .env
```
